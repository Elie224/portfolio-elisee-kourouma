Utilisation
- Ouvrez `index.html` dans votre navigateur (double-clic) ou servez le dossier avec un serveur statique.
  - Exemple (PowerShell): `python -m http.server 8000` puis aller sur `http://localhost:8000`

Structure
```
index.html
about.html
projects.html
contact.html
assets/
  css/
    styles.css
  js/
    main.js
```

Personnalisation
- Modifiez les pages `.html` pour le contenu.
- Ajoutez vos projets dans `projects.html` (tableau JS inline) ou chargez un fichier JSON statique si besoin.
